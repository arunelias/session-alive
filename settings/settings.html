Background Request Advanced Options Optional
Some websites implements JavaScript idle timer in browser also. If you know how the JavaScript works in the page, executing a small JavaScript code will reset the idle timer, for eg: leftTime=20; idleTime=0; clearTimeout(idle-timer); The supplied JavaScript code is inserted to the page and executed. This option requires a minimum knowledge of web programming to be able to properly use it.

The supplied JavaScript code is inserted to all pages with URL starting with the Trigger URL. For example Trigger URL is set to https://example.com/user/, the supplied JavaScript code is inserted and executed to pages https://example.com/user/profile or https://example.com/user/account/details on any tab of the browser window.

Foreground Request Options Optional
Valid trigger URL (including "http://" or "https://") is required.
Some websites implements JavaScript idle timer in browser also. For this websites requesting the page in background will not be enough to keep the session alive. The page needs to reload to keep the session. Navigating to a any URL starting with the Trigger URL will start the foreground page reload loop. For example Trigger URL is set to https://example.com/user/, navigating to https://example.com/user/profile or https://example.com/user/account/details will set to auto reload in the set interval
Valid Foreground Auto-Reload Interval (in minutes) is required.
Foreground Auto-Reload interval for the foreground auto-reload loop. Minimum 1 minute and whole numbers only.
Play beep sound before reloading page.
